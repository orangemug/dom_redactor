Builds a skeleton view of DOM content, the intended use is for displaying content to be bought whilst remaining the structure of the content

Usage

There are two steps firstly processing the content, this should be done server side using node.js and can be accomplished in the following way

var dom = document.getElementById('content') DOMSkeleton.process(dom)

Secondly rendering the content

var dom = document.getElementById('content') DOMSkeleton.render(bodyDOM)
